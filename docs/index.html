What this is

This is not a final permit opinion and not a construction quote form. It is an early property screen that helps separate cleaner-looking projects from blocker-heavy cases, older unpermitted work, or situations that need deeper records review. The first job is usually confirming the address, city path, and issue type before anyone starts talking like every property belongs in the same process.

这不是最终 permit 结论，也不是装修报价页。它更像一个前置分诊工具，先把地址、城市入口、 项目类型和 blocker 信号看清楚，再决定该继续走标准路径、先做 blocker diagnosis， 还是按救援 / 合法化思路处理。

Built for real homeowner questions

Most people do not start with a clean permit strategy. They start with one of these questions: “Can I convert this garage?”, “Is this lot even workable?”, or “What do I do about an old unit that was built without permits?”

Why address and city come first

很多客户一开口就问 “能不能做” 或 “大概多少钱”，但真正更早、更值钱的问题通常是： 这个地址到底属于 LA City、LA County，还是 Arcadia、Pasadena、San Gabriel、Monterey Park 这样的独立城市。 只要 jurisdiction 判断错，后面整条路径都会错。

LA City is not the same as LA County

Mailing address alone does not tell you the real path. A property can say Los Angeles and still belong to County or a different city workflow.

邮寄地址写 Los Angeles，不等于一定属于 LA City。很多客户实际归 County 或其他城市系统。

SGV cities do not use one shared playbook

Arcadia, San Gabriel, Monterey Park, and Pasadena each have their own entry points, review style, and friction patterns.

圣盖博谷各城市入口和审核逻辑不同，不能把 Arcadia 的经验直接套到 Pasadena 或 San Gabriel。

Path names depend on the real situation

Not every existing issue is the same thing. Some properties fit a cleaner ADU path, some need blocker review, and some move like rescue or compliance cases.

不是每个现有问题都叫合法化，也不是每个旧车库都适合直接讲 ADU 路径。先看现状和记录，再决定叫法。

That is why the screen starts with address, project type, and blocker signals before it talks about next steps.
