Before & After Visuals 前后对比素材

Representative project visuals for the kind of upgrade direction homeowners usually want to see first.

这些视觉素材帮助业主先看懂“升级方向”是什么样，而不是先把路径想成已经确定。

These visuals help customers picture finish level and improvement direction. They do not replace address screening, jurisdiction review, or project-specific feasibility. The point is to help a homeowner see the gap between current condition and a cleaner finished outcome before asking which path the property can really support.

这些素材帮助客户理解 finish level 和升级方向。它们不能替代地址预筛、jurisdiction 判断或项目可行性分析。 目的只是让业主先看到“当前状态”和“更干净结果”之间的差距，再去判断物业到底适合哪条路径。

How to use this page

怎么使用这个页面

Use this for visual reference

把这里当成视觉参考

It helps homeowners and agents show what a cleaner finish direction can look like when they are discussing whether a project feels worth exploring.

它帮助业主和 agent 在讨论项目值不值得继续时，更直观地看到“更干净的完成方向”是什么样。

Do not use this as a construction promise

不要把它当成施工承诺

Older records, city path, legal status, and site conditions still decide whether a property should move toward garage conversion, detached ADU exploration, or a heavier existing-issue review.

旧 records、城市入口、合法状态和现场条件，仍然决定了这个物业到底更像车库改造、独立 ADU 探索，还是更重的现有问题审核。

What should happen next

下一步应该做什么

  • Use the visuals to anchor the homeowner’s expectations.
  • Then screen the address before anyone quotes scope or promises a path.
  • If the property shows city or records friction, move into review instead of pretending every property is a clean standard case.
  • 先用这些素材帮业主建立对结果方向的直观预期。
  • 再去做地址预筛，不要先报 scope 或承诺路径。
  • 如果物业已经暴露城市或 records friction，就应该进入 review，而不是假装每个物业都是标准干净案子。
